# kiosk
aplikacja kiosków samoobsługowych dla gastronomii / restauracji


Instalacja:

<code>wget -O - https://raw.githubusercontent.com/ciasther/kiosk/main/test-install.sh | sudo bash</code>

<code>wget -O - https://raw.githubusercontent.com/ciasther/kiosk/main/kiosk-install.sh | sudo bash</code>

<code>wget -qO- https://raw.githubusercontent.com/ciasther/kiosk/main/kiosk-install.sh | sudo bash -s kiosk</code>

<code>wget -qO- https://raw.githubusercontent.com/ciasther/kiosk/main/kiosk-install.sh | sudo bash -s cashier</code>

<code>wget -qO- https://raw.githubusercontent.com/ciasther/kiosk/main/kiosk-install.sh | sudo bash -s display</code>
